其实很早就听说Python，但是一直没怎么学，主要是不太了解能干嘛。之前在一个骐达的初级课程中就听说过PHP及Python，

看过那个教程之后，总想有朝一日也能搞一个牛叉的爬虫出来——目的很简单，就是把我们查询任务的东西爬下来或者把我们查询天气的那个爬下来，
